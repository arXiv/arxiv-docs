# Truetype and Opentype fonts available at arXiv's TeX Live 2025

The following list gives font names and file names for all
truetype and opentype fonts available in TeX Live 2025.

Usage of these fonts in LaTeX documents using xelatex can be done
by specifying the **file name**. Lookup via font name is currently
not supported.
